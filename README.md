# Iris Prediction

This repository contains a Jupyter Notebook implementation for predicting the species of iris flowers using machine learning techniques. The classic Iris dataset, which includes measurements for sepal length, sepal width, petal length, and petal width for three species of iris (setosa, versicolor, and virginica), is used for model training and evaluation.

## Features

- Data exploration and visualization
- Data preprocessing and cleaning
- Building and training machine learning models (e.g., Logistic Regression, Decision Tree, etc.)
- Model evaluation and accuracy metrics
- Predicting iris species based on user input

## Dataset

The Iris dataset is a well-known multivariate dataset introduced by Sir Ronald Fisher in 1936. It contains 150 samples from each of three species of Iris (setosa, versicolor, and virginica).
